===============================================================
HMRV RAG 智慧查詢系統 - 快速開始指南
===============================================================

你已經成功建立了 RAG 知識庫系統！

接下來請按照以下步驟操作：

【步驟 1】安裝 Python 環境
   雙擊執行：1_install.bat
   └─ 會自動建立虛擬環境並安裝所有必要套件

【步驟 2】安裝 Ollama 和下載 Embedding 模型
   1. 前往 https://ollama.com 下載並安裝 Ollama
   2. 雙擊執行：2_setup_ollama.bat
   3. 選擇 1（qwen3-embedding，推薦）

【步驟 3】設定你的區網 LLM API
   編輯 config.yaml，修改以下設定：

   llm:
     base_url: "http://你的IP:你的Port/v1"
     api_key: "如果需要就填入"
     model: "qwen3-30b"  # 依你的 API 填寫

【步驟 4】建立知識庫（需要 20-40 分鐘）
   雙擊執行：3_build_db.bat
   └─ 會掃描專案的 722 個 Java 檔案並建立向量資料庫

【步驟 5】開始查詢
   方式 A（推薦）：雙擊執行 5_web_ui.bat
   方式 B：雙擊執行 4_query.bat（命令列版）
