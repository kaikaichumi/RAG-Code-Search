## 從 v1.0 升級到 v2.0

如果你使用的是舊版本（v1.0），以下是升級步驟：

### 1. 備份舊的知識庫（可選）

```bash
# 如果你想保留舊的知識庫
cp -r chroma_db chroma_db_backup
```

### 2. 更新程式碼

```bash
git pull
```

### 3. 更新依賴套件

```bash
# 啟動虛擬環境
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 更新套件
pip install -r requirements.txt
```

### 4. 更新設定檔

編輯 `config.yaml`，移除 `include_dirs` 欄位（現在改為執行時選擇）

### 5. 重新建立知識庫

```bash
# 使用新的建置工具
python rag_builder.py
```

### 6. 開始使用

```bash
# Web UI
python rag_web_ui.py

# 或命令列
python rag_query.py
```
